# World-Cup-2026-Prediction-Model
Oiiiee!! Estou tentando prever os jogos da copa desse ano

Primeiros passos (25/06/2026)

Para começar, estou treinando meu modelo com base em todos os jogos realizados ao longo da história desde 1872, entre as seleções que participam da Copa do Mundo de 2026.
Tambem estou levando em conta eventuais mudanças nos nomes dos países.
Além disso, estou utilizando apenas dados anteriores ao início da Copa do Mundo deste ano (2026), para que eu possa usar as partidas que já aconteceram como testes de precisão.

Hello! I'm trying to preedict the world cup games

First steps (06/25/2026)

First I am starting to train my model using all of the past games between teams currently participating in the world cup, starting from 1872.
I'm accounting for name changes in countries just in case there is any btw.
Also, I'm only using data up to before the world cup starting this year (2026), so that I can use the games that already happeneed as tests of accuracy.
